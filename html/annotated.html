Lista de clases
Lista de las clases, estructuras, uniones e interfaces con una breve descripción:
 CJuegoClase que proporciona métodos estáticos, para el funcionamiento del juego
 CListaSirve para la representación de montones y de la cinta numérica. Lista[naturales] mediante celdas doblemente enlazadas
 ClistaMontonesEstructura para almacenar los montones del juego. Almacena objetos de la clase Lista: listaMontones[Lista]
 Cnodo_ListaRepresenta la celda de una lista, del tipo Lista[naturales], una lista en la que cada celda contiene una número natural
 Cnodo_listaMontonesRepresenta la celda de una lista, listaMontones[Lista], una lista en la que cada celda contiene una lista de números
 CPrincipalClase empleada para interactuar con el usuario y para crear objetos de las clases lista y listaMontones
